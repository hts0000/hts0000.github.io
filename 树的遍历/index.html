树的遍历方式分为如下几种：

  • 前序遍历
  • 中序遍历
  • 后序遍历
  • 层序遍历

其中前序遍历可以求出树的深度，而后序遍历可以求出树的高度

树的遍历也可以借用辅助数据结构来完成：

  • 栈
  • 队列

栈适合模拟前中后序遍历，队列适合模拟层序遍历

func maxDepth(root *TreeNode) int {
	if root == nil {
		return 0
	}
	leftDepth := maxDepth(root.Left)
	rightDepth := maxDepth(root.Right)
	return max(leftDepth, rightDepth) + 1
}
